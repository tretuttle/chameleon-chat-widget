Amigo Widget Integration Example

This page demonstrates how to integrate the Amigo support widget into any website.

Integration Methods

Method 1: Auto-initialization

Add the data attribute to automatically initialize the widget when the page loads:

<link rel="stylesheet" href="https://yoursite.com/amigo-widget.css"> <script src="https://yoursite.com/amigo-widget.umd.js" data-amigo-widget="auto"></script>

Method 2: Manual initialization

Initialize the widget programmatically with custom configuration:

<link rel="stylesheet" href="https://yoursite.com/amigo-widget.css"> <script src="https://yoursite.com/amigo-widget.umd.js"></script> <script> AmigoWidget.init({ containerId: 'my-custom-container', theme: 'light' }); </script>
